GildedRose_Ruby
==================

Intro
-------
My OOD solution to the Ruby version of the GildedRose kata.

Technologies used
-------
Development: Ruby
Test: Rspec, Simplecov

How to install
-------
1. Clone the repo
2. run 'bundle' in the project folder

How to use
-------
in command line
'ruby texttest_fixture.rb' for a quick texttest.

Author
